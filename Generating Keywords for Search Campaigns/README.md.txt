# Generating Keywords for Search Campaigns

## 📌 Objective
Create a structured set of targeted keywords for a digital marketing campaign focused on promoting affordable furniture products, specifically sofas and related items.

## 🛠 Tools Used
- Python
- pandas

## 📊 Problem Context
A furniture retailer wanted to run search campaigns targeting price-sensitive customers. The goal was to generate relevant keywords emphasizing affordability, discounts, and promotions.

## 🔍 Process
- Defined product categories (e.g., sofas, recliners, sofa beds)
- Created a list of marketing-focused keywords (e.g., buy, discount, promo)
- Combined products and keywords programmatically
- Structured the output into a DataFrame suitable for campaign setup
- Exported results to a CSV file for use in advertising platforms

## 📈 Key Insights
- Programmatically generating keywords ensures scalability for large campaigns
- Focusing on price-related terms aligns with the target audience (budget-conscious customers)
- Structured keyword datasets can be directly integrated into SEM tools

## 🚀 Outcome
Produced a clean and scalable keyword dataset that can be used to launch targeted search engine marketing campaigns efficiently.

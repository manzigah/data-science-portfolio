# Analyzing Crime Patterns in Los Angeles

## 📌 Objective
Analyze crime data from Los Angeles to identify patterns in criminal activity and support better resource allocation for law enforcement.

## 🛠 Tools Used
- Python
- pandas
- numpy
- matplotlib
- seaborn

## 📊 Dataset
- Crime dataset from Los Angeles including time, location, victim demographics, and crime details

## 🔍 Process
- Cleaned and prepared the dataset
- Extracted hour of occurrence from time data
- Identified peak crime hours
- Analyzed nighttime crime patterns by location
- Grouped victim ages into meaningful brackets

## 📈 Key Insights
- Crime peaks around midday (12:00)
- The Central area has the highest number of nighttime crimes
- Most victims fall within the 26–34 age group

## 🚀 Outcome
This analysis provides actionable insights that can help law enforcement agencies allocate resources more effectively and better understand crime patterns.
